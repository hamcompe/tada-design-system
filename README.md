# Tada Design System

Simple, reusable, documented components.

This project has been built by,

- [Next.js](https://github.com/zeit/next.js/) for usage testing
- Styling by [emotion-js](https://github.com/emotion-js/emotion)
- Documented by [docz](https://github.com/pedronauck/docz)

## Setup

Run the following command

```shell
yarn
```

## Development

Run the following command

```shell
yarn docz:dev
```
